# UI Kit · Dashboard de Monitoramento de Programa

Painel executivo para acompanhamento de programa estratégico (caso de uso: Moderniza-ES / BID).

## Estrutura
- `index.html` — entrada
- `App.jsx` — composição
- `Sidebar.jsx` — navegação lateral institucional
- `ProgramHeader.jsx` — header com nome do programa, status, breadcrumbs
- `KpiTile.jsx` — bloco KPI reutilizável
- `IndicatorTable.jsx` — tabela de indicadores com status
- `MilestonesPanel.jsx` — marcos do cronograma
- `BudgetChart.jsx` — barra de execução orçamentária
