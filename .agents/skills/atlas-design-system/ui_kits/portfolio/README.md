# UI Kit · Portfólio · Ariadina Astori Porto

Recriação em alta fidelidade do site oficial <https://ariadina-astori.com.br/>.

## Estrutura
- `index.html` — entrada (React + Babel + Lucide + componentes)
- `App.jsx` — composição da página
- `Header.jsx` — header fixed transparente (vira navy+blur ao rolar)
- `Hero.jsx` — hero com gradiente navy + ilustração de globo + stats
- `AboutSection.jsx` — Sobre (monograma + descrição)
- `ExperienceSection.jsx` — Experiência (timeline)
- `EducationSection.jsx` — Formação (3 cards)
- `SkillsSection.jsx` — Habilidades (chips agrupadas)
- `ContactSection.jsx` — Contato (gradient navy + formulário)
- `Footer.jsx` — Rodapé minimalista
- `portfolio.css` — Estilos completos

## Como usar
Abra `index.html`. Tokens vêm de `../../colors_and_type.css`.
